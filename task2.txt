JSON - это независимый от языка реализации формат для обмена данными в виде текста, удобный для чтения человеком. 
    JSON подходит для сериализации сложных структур. Применяется для обмена данными как между браузером и сервером, так и между самими серверами.
    Так же Json можно использовать для файлов конфигурации.

YAML - формат сериализации данных. В основном используется как формат для файлов конфигурации. Применяется для настройки веб-каркасов. 
    Также является основным языком описания классов, ресурсов и манифестов для пакетов приложений.

XML - это расширяемый язык разметки (Extensible Markup Language), разработанный специально для размещения информации в World Wide Web
    Представление документов любого (не только текстового) типа, например, музыки, математических уравнений и т.д.
    Сортировка, фильтрация и поиск информации.
    Представление информации в структурированном (иерархическом) виде.
